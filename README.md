# 🎓 ❀工学术英语(下)期末复习系统

这是一个专为 **学术英语(下)期末备考** 量身定做的自适应智能复习系统。

本复习系统收录了学术英语（下）期末考试可能遇到的 **所有原题及原文（包括听力和阅读）**，并全部附上 **中英文对照翻译及原题正确答案**，且提供诸如 **词汇刷题** 等实用功能，可以帮助考生全面复习。

本系统采用 **纯前端单文件（HTML + CSS + JS）** 设计，配以炫酷的深色科技风 UI 界面。您无需安装任何依赖或配置服务器，**只需直接双击 [学术英语期末复习系统.html] 在任意浏览器中打开即可开始复习**。

---

## 🎯 四大核心备考板块 (大纲配比)

根据官方考试大纲，本系统将复习划分为以下四大板块，精准对应期末分值：

### 1. 🎧 听力训练板块 (占比 20%)
* **内容范围**：下册 1-5 单元 **Viewing 视频（5 篇）** 与视听说3 **Further Listening 听力短文（10 篇）**（皆为考试原题/重点）。
* **功能特色**：
  * **中英对照**：提供逐句的中英文翻译对照。
  * **盲听模式**：支持一键“折叠中文”，实现纯英文盲听训练。
  * **配套强化**：内嵌针对性多选测试，并配有详细的解析。

### 2. ✍️ 词汇刷题板块 (占比 30%)
* **内容范围**：涵盖《通用学术英语短语词汇教程》子表 6—10 的全部 **270 道单项选择题**。
* **功能特色**：
  * **背题模式**：一键开启。开启后正确选项显示为绿色，译文与干货解析直接呈现，并高亮下划线标注学术关键词，方便快速浏览记忆。
  * **生词收藏**：点击题目右上角的 `☆` 可加入收藏夹，结合“**收藏夹**”过滤器进行重点突击。
  * **进度拖拽**：题卡上方配有滑块进度条，支持直接点击或左右拖拽，轻松跳转到任意题号。

### 3. 📖 阅读精读板块 (占比 30%)
* **内容范围**：综合教程 1-5 单元的 10 篇精选课文 + 智慧阅读 6 篇课文（期末考试阅读 3 篇的重要来源）。
* **功能特色**：
  * 支持左右分栏双语对照阅读。
  * 提取出每篇文章的**学术高频词汇包**与**长难句核心句法结构剖析**（如主谓宾、复杂从句拆解）。
  * 包含针对性设计的 AI 模拟单选题。

### 4. 📝 写作范文板块 (占比 20%)
* **内容范围**：针对 1-5 单元核心主题的 **5 篇高分学术范文**，完全对标雅思学术风格（150-200 词）。
* **功能特色**：
  * 提炼了范文中的**核心写作框架**与高档学术句式。
  * 整理了高频学术词汇搭配，便于快速背诵和考场套用。

## 🧠 自适应智能算法说明 (词汇专属)

* **连续答对排除算法**：当某道题被您连续答对若干次时（系统默认值为 **2 次**，您可在控制栏自行指定此排除阈值），系统判定您已牢固掌握。该题在进行「随机抽查（随机抽取 60 题）」时会自动被屏蔽，避免在已掌握的熟词上浪费时间。
* **错题动态加权排行**：系统会精确统计您在每道题上的答错频率。当点击「错题排行」按钮时，系统会重新排序，将您答错次数最多的题目排在最前列，实现高频易错词强化。

## ⌨️ 词汇刷题快捷键

在词汇刷题板块中，可以使用以下键盘快捷键进行高效率刷题：

| 快捷键 | 功能 | 说明 |
| :--- | :--- | :--- |
| **`Space` (空格)** 或 **`D`** | **下一题 / 跳过** | 切换到下一题，不记为答错 |
| **`A`** | **上一题** | 返回到上一道题 |
| **`1` / `2` / `3` / `4`** | **选择选项** | 分别对应 A / B / C / D 选项 |

> [!NOTE]
> 当用户的输入光标聚焦在文本框（如修改屏蔽次数）或者阅读弹窗打开时，快捷键会自动挂起以防冲突。

## ⚙️ 系统辅助功能

* **无级字体大小调节**：页面右下角配有字号微调面板，支持在 `80%` 到 `150%`（12px - 24px）之间任意缩放页面，完美适配手机、平板与电脑屏幕。
* **本地存档持久化**：系统基于浏览器 `LocalStorage`，你的做题记录、答对/答错统计、收藏夹状态皆保存在本地，关闭网页或刷新不会丢失进度。
* **数据重置归零**：若在复习完一轮后想重新清零测试，可在词汇控制栏点击 `🗑️ 重置统计`，一键清除所有答题记录与错题权重。
